### Images

Artist sizes:
* lg - original size ~10-20% compression
* md - original size ~30% compression
* sm - 900x600 px + compressed

Social media images:

* Instagram - 640w option
* Twitter - small option

### Env vars

* PORT - eg.: 3000
* ARTIST - eg.: danika-mori
* GO111MODULE - eg.: on

### Running the application
```
go run main.go
```
